# Walmart Store Sales Performance & Holiday Impact Analysis
A data analysis project exploring weekly sales across 45 Walmart stores using Excel, SQL and Power BI.
## Tools Used
- Excel
- SQL
- Powe BI
## Dashboard Features
- Weekly Sales Trend Analysis
- Holiday vs Non-Holiday Sales Comparison
- Top & Bottom 10 store Performance
- Temperature vs Weekly Sales Analysis
- Fuel Price vs Weekly Sales Analysis
- Interactive Store, Year and Holiday Filters

## Conclusion
Walmart's sales performance is driven primarily by seasonal demand rather than external factors such as temperature or fuel prices.
Holiday periods consistently generated stronger sales, while a small group of stores outperformed the rest of the network.
These insights highlight the importance of strategic holiday planning and stor-level maximizing revenue.
